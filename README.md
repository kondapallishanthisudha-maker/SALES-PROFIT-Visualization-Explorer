# 📊 SALES-PROFIT-Visualization-Explorer

An interactive data visualization application built using **Python**, **Matplotlib**, and **Gradio** that helps users explore sales and profit trends through various charts and graphical insights.

## 🚀 Features

- Upload sales dataset (CSV format)
- Visualize Sales vs Profit relationships
- Generate bar charts, line plots, scatter plots, and histograms
- Interactive web interface using Gradio
- Quick business insights through visual analysis

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Gradio
